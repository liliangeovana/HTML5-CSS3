Principais formatações

Negrito / Destaque

Nesta frase temos um termo em negrito utilizando a tag <b>

Nesta frase temos um termo em destaque utilizando a tag < strong >

Itálico / Ênfase

Nesta frase temos um termo em itálico utilizando a tag <i>

Nesta fase temos um termo em ênfase usando a tag < em >

Texto Marcado

Podemos criar também um texto marcado usando a tag MARK

Neste outro parágrafo temos outro texto marcado no final.

Texto grande e texto pequeno

Estamos criando um texto pequeno com a tag <small>

Para caracterizar um texto grande é necessário CSS; a tag <big> já está obsoleta.

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado.

Texto inserido

Podemos marcar um texto como inserido para dar uma enfâse para indicar que ele foi adicionado depois.

Existe também o sublinhado com a tag U (não semântica)

Texto sobrescrito

Para inserir coisas do tipo: x^2+3 ==> x2+3 tag <sup>

Texto subscrito

Para inserir coisas do tipo: H2O ==> H2O tag <sub>
